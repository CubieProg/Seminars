# Seminars

---

### Общая информация

Файлы семинаров по курсу "Алгебра и геометрия", НГУ, ФИТ, гр. 25209.

Репозиторий устроен следующим образом:
- В папке **Literature** находится вся рекомендованная литература по данному курсу;
- В папке **Semester I** находятся (на самом деле ещё нет) файлы по первому семестру;
- В папке **Semester II** находятся файлы по второму семестру;
- **README** - и есть README. 


---

### Обновление репозитория

Примерно за день до семинара будет происходить обновление репозитория в два этапа:
1. Будет коммит двух новых файлов в папку **Semester II**. Файлы будут названы соответствующей датой;
2. Будет выпуск нового релиза с тегом, который называется соответствующей датой.



**Как найти актуальные файлы?**
1. Прийти на пару;
2. Открыть этот репозиторий;
3. Просмотреть последний релиз;
4. Файлы находятся в прикреплённых бинарниках. (В архивах _"Source Code.zip"_ и _"Source Code.tar.gz"_ лежит **весь** репозиторий целиком. Скачивать их **не нужно**.

**или**
1. Прийти на пару;
2. Открыть этот репозиторий;
3. Открыть папку **Semester II**;
4. Искать файлы с нужной датой.
